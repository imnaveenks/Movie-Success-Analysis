# What Makes a Movie Successful? A Data Analysis

## Overview
This project explores what factors drive box office success using the IMDB Top 1000 dataset. 
The analysis was performed in Python (Google Colab) using pandas and matplotlib.

## Dataset
- **Source:** Kaggle - IMDB Top 1000 Movies
- **Size:** 1000 movies, cleaned down to 831 with complete gross revenue data

## Key Questions
- Which genres generate the highest box office revenue?
- Do higher IMDB ratings lead to higher earnings?
- Which directors consistently deliver the highest grossing films?

## Key Findings
- **Family and Action films** generate the highest average box office revenue
- **High IMDB ratings don't guarantee commercial success** — the highest grossing movies cluster around 7.75–8.5, not 9+
- **Anthony Russo and J.J. Abrams** are the top directors by average gross revenue
- **Star Wars: Episode VII** is the highest grossing film in the dataset at $936M
